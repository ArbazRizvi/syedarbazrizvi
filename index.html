Kids Guard Application (FYP)

Project Overview

'Kids Guard' was my Final Year Project (Grade A), designed as a comprehensive parental control and child safety application. The primary goal was to provide parents with tools to monitor their child's digital activity and ensure their safety from online threats like cyberbullying, explicit content, and communication with strangers.

Tech Stack

Android (Java) Firebase Realtime DB Firebase Authentication Python (for ML Model) Scikit-learn

Key Features

  • Real-time Risk Detection: Implemented a Machine Learning model (trained on a custom dataset) to scan SMS and app notifications for keywords related to bullying, depression, and explicit content.
  • Parental Dashboard: A secure portal for parents to view alerts, see app usage statistics, and manage settings.
  • App Blocker: Allowed parents to block or set time limits on specific applications.
  • SOS Alerts: A dedicated SOS button for the child to immediately alert parents with their location.
